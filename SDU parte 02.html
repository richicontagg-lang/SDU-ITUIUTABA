      ● IFTM · Projeto Integrador Extensionista III

      Sistema de Denúncias Urbanas
      Inteligentes de Ituiutaba

      Uma plataforma de software livre para conectar cidadãos à prefeitura, simplificando o registro e acompanhamento de problemas urbanos.

      Licença MIT Software Livre Extensão Universitária Ituiutaba · MG ADS · 2026
      Role para saber mais

      O que é este projeto?

      Ituiutaba enfrenta problemas urbanos recorrentes — buracos nas vias, iluminação pública falha e descarte irregular de lixo — sem um canal eficiente de comunicação entre moradores e poder público. Este sistema propõe uma solução: uma plataforma acessível, baseada em software livre, onde qualquer cidadão pode registrar uma denúncia com foto, geolocalização e acompanhar a resolução em tempo real.

      Sobre o Projeto

      Contexto, comunidade e proposta de valor

      🏙️

      Comunidade atendida

      Moradores de Ituiutaba–MG (~105 mil hab.), comerciantes locais e a Prefeitura Municipal, afetados por problemas de infraestrutura urbana.

      🎓

      Extensão universitária

      Projeto desenvolvido no IFTM Campus Ituiutaba como Projeto Integrador Extensionista III, aplicando conhecimento acadêmico na resolução de problemas reais.

      📜

      Licença MIT

      Software livre com licença MIT: qualquer cidade pode adaptar, modificar e redistribuir o sistema. Código aberto no GitHub para colaboração contínua.

      🎯

      Problemas mapeados

      Buracos nas vias, falha na iluminação pública e descarte irregular de lixo identificados como principais problemas via pesquisa com moradores.

      📈

      Indicadores de impacto

      Redução no tempo de resposta da prefeitura, aumento no número de problemas resolvidos e maior participação da população.

      🤝

      Estratégia de colaboração

      Parceria com a Prefeitura, entrevistas com moradores, testes com usuários reais e publicação no GitHub para evolução contínua.

      Tecnologias Utilizadas

      Stack 100% software livre

      ⚛️

      React.js

      Interface web moderna e responsiva com componentes reutilizáveis.

      🟢

      Node.js + Express

      API REST para gerenciar denúncias, usuários e notificações.

      🐘

      PostgreSQL

      Banco de dados relacional robusto com suporte a dados geoespaciais via PostGIS.

      🗺️

      OpenStreetMap

      Mapas gratuitos e de código aberto para exibir geolocalização das denúncias.

      📱

      PWA

      Progressive Web App: funciona como aplicativo instalável no celular, com notificações push.

      🐙

      GitHub

      Repositório público para versionamento, colaboração e transparência.
