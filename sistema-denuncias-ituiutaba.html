Role para ver o protótipo

O que é este projeto?

Ituiutaba enfrenta problemas urbanos recorrentes buracos nas vias, iluminação pública falha e descarte irregular de lixo sem um canal eficiente de comunicação entre moradores e poder público. Este sistema propõe uma solução: uma plataforma acessível, baseada em software livre, onde qualquer cidadão pode registrar uma denúncia com foto, geolocalização e acompanhar a resolução em tempo real.

Sobre o Projeto

Contexto, comunidade e proposta de valor

🏙️

Comunidade atendida

Moradores de Ituiutaba–MG (~105 mil hab.), comerciantes locais e a própria Prefeitura Municipal, diretamente afetados por problemas de infraestrutura urbana.

🎓

Extensão universitária

Projeto desenvolvido no IFTM Campus Ituiutaba como Projeto Integrador Extensionista III, aplicando conhecimento acadêmico na resolução de problemas reais da comunidade.

📜

Licença MIT

Software livre com licença MIT: qualquer cidade pode adaptar, modificar e redistribuir o sistema. Código aberto no GitHub para colaboração contínua.

🎯

Problemas mapeados

Buracos nas vias, falha na iluminação pública e descarte irregular de lixo foram identificados como os principais problemas via pesquisa com moradores.

📈

Indicadores de impacto

Redução no tempo de resposta da prefeitura, aumento no número de problemas resolvidos e maior participação da população no cuidado com a cidade.

🤝

Estratégia de colaboração

Parceria com a Prefeitura, entrevistas com moradores, testes com usuários reais e publicação no GitHub para garantir evolução contínua do sistema.

Tecnologias Utilizadas

Stack 100% software livre

⚛️

React.js

Interface web moderna e responsiva. Componentes reutilizáveis e atualizações em tempo real sem recarregar a página.

🟢

Node.js + Express

API REST para gerenciar denúncias, usuários e notificações. Leve e escalável para o porte do projeto.

🐘

PostgreSQL

Banco de dados relacional robusto para armazenar denúncias, usuários e histórico de status. Suporte a dados geoespaciais via PostGIS.

🗺️

OpenStreetMap

Mapas gratuitos e de código aberto para exibir geolocalização das denúncias sem depender de APIs pagas.

📱

PWA

Progressive Web App: funciona como aplicativo instalável no celular, com notificações push e acesso offline parcial.

🐙

GitHub

Repositório público para versionamento, colaboração e transparência. Qualquer desenvolvedor pode contribuir.
